---
title: "HHS/USDA Select Agent Regulations Summary"
source: "42 CFR Part 73 (HHS), 7 CFR Part 331 (USDA), 9 CFR Part 121 (USDA)"
---

The HHS and USDA Select Agent Regulations (SAR) define the list of biological agents and toxins that pose the greatest risk of deliberate misuse and require special handling, registration, and security measures. HHS Tier 1 select agents include: Variola major (smallpox), Bacillus anthracis (anthrax), Botulinum neurotoxin (all serotypes above 0.5 mg quantities), Francisella tularensis (tularemia), Yersinia pestis (plague), Ebola virus, Marburg virus, and several others. Toxin thresholds are defined by quantity: botulinum toxin is select agent-regulated above 0.5 mg. The Select Agent Program imposes: registered facility requirements, personnel reliability reviews, inventory tracking, security plans, and incident reporting. From a protocol screening perspective, the SAR defines the set of reagents and agents that trigger mandatory regulation regardless of workflow intent. Any protocol involving a listed select agent or toxin at threshold quantity triggers SAR compliance requirements independently of biosecurity screening.
